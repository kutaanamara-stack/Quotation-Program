# 餐具修复报价工具使用说明

## 推荐入口

电脑端请双击：

`start-webapp.cmd`

手机端请直接打开公网链接：

`https://kutaanamara-stack.github.io/Quotation-Program/`

## 重要说明

不要直接双击 `webapp/index.html` 或 `网页版/index.html`。这类网页需要通过本地小服务或公网链接打开，直接双击文件时，部分浏览器会显示空白页。

## 电脑端使用步骤

1. 双击 `start-webapp.cmd`
2. 等浏览器自动打开报价工具
3. 使用期间保持启动窗口打开
4. 用完后关闭启动窗口即可

如果电脑提示没有 Python，请改用手机公网链接，或联系管理员安装 Python 3。

## 手机端使用步骤

直接在手机浏览器打开：

`https://kutaanamara-stack.github.io/Quotation-Program/`

手机端不需要连接公司 Wi-Fi。只要 GitHub Pages 已发布成功，手机使用流量也可以打开。

## 报价操作

1. 输入客户公司名称
2. 填写固定规格对应的数量
3. 需要统一调价时打开“高级设置”，修改统一系数
4. 某一行需要单独改价时，在“高级设置”里启用该行手动单价
5. 检查预览和合计
6. 点击“导出图片”或“导出 PDF”

## 常见问题

如果电脑端打开空白页，请确认是否双击了 `start-webapp.cmd`，不要直接打开 `webapp/index.html`。

如果手机公网链接打不开，请联系管理员确认 GitHub 仓库的 `Settings -> Pages` 是否选择了 `GitHub Actions`，并确认 `Actions` 发布任务已经变成绿色对勾。
