## 使用步骤

1. 电脑双击 `启动网页版.cmd`
2. 浏览器会自动打开工具页面
3. 如需手机访问，请让手机和电脑处在同一 Wi-Fi 下，并使用脚本窗口里显示的局域网地址
4. 在页面中填写 `客户公司名称`
5. 按需要填写每一行的 `白胚数量`
6. 如需统一调价，展开 `高级设置` 修改 `统一系数`
7. 如某一行需要单独改价，在 `高级设置` 中勾选对应行的 `手动改单价`
8. 检查右侧/下方预览
9. 点击 `导出图片` 或 `导出 PDF`

## 适用说明

- 这是一个本地网页版工具，不需要安装
- 建议使用 `Edge` 或 `Chrome`
- 可以直接从共享文件夹中打开
- 不建议直接双击 `网页版/index.html`，推荐使用 `启动网页版.cmd`
- 启动本地网页版需要电脑里有 `Python 3`（本机已验证可用）
- 手机端最终需要使用上面的 GitHub Pages 链接打开
- 如果手机端链接还是 `404`，请先在 GitHub 仓库里启用 Pages / 等待 Actions 发布完成
